# ADR 0003: Curated artwork from v1

## Status

Accepted

## Context

v1 ships many oil and pencil images. Dumping all into masonry weakens the brand-home experience.

## Decision

Reuse v1 assets as the source pool, but ship a **curated subset** per gallery (oils, sketches) plus hero/profile imagery.

## Consequences

Faster pages, stronger first impression. More pieces can be added later without redesign.
